## Install

Run in console

```sh
    pip3 install -r requirements.txt
```

## Set Up

Copy and paste _.env.example_, rename it with _.env_, and set the data required

## Run the project

### Windows

`py manage.py runserver`

### Linux & Mac

`python3 manage.py runserver`
